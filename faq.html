General

EyaWriter is a browser-based screenplay writing application with built-in AI assistance, real-time collaboration, and one-click DOCX export. It handles professional screenplay formatting automatically so you can focus on your story.
You can explore EyaWriter without an account using the demo mode. To save your work to the cloud, sync across devices, or collaborate, you'll need a free account — which takes about 30 seconds to create.
EyaWriter runs in any modern browser, including mobile browsers. For extended writing sessions, a desktop or tablet with a keyboard is recommended. A dedicated mobile app is on the roadmap.
No installation required. EyaWriter runs entirely in your browser. It also supports offline editing through a Progressive Web App (PWA) — your changes sync automatically when you reconnect.
EyaWriter uses the standard Hollywood screenplay format: Courier 12pt font, 1.5-inch left margin, 1-inch other margins, with proper element spacing for scene headings, action lines, character names, dialogue, and parentheticals. This matches the WGA and industry-standard format expected for submissions.

AI Features

EyaWriter connects to a language model API (via OpenRouter) to power features like grammar correction, dialogue improvement, and smart proofreading. Requests are sent from your browser to our proxy server, then to the AI provider. Your script content is only sent for the specific line or scene you've highlighted.
Yes. Go to Settings → AI → enter your API endpoint URL. This bypasses our shared rate limits entirely, giving you unlimited AI use with your preferred model. OpenRouter, OpenAI, and Anthropic-compatible endpoints are supported.
Only when you explicitly trigger an AI action. EyaWriter sends the selected text (and a small surrounding context window) to the model. We never batch-process or background-scan your scripts. We do not use your content to train any models.
The default shared endpoint uses a capable instruction-following model via OpenRouter. When you bring your own API key, you can target any model available on that platform — including GPT-4o, Claude, and others.

Collaboration

Open your script, go to the Collaborate panel in the toolbar, and enter your collaborator's email address. They'll receive an invitation. They'll need a free EyaWriter account to accept and edit.
Owner — full control, can invite/remove collaborators and delete the project. Editor — can read and write script content and comments. Viewer — read-only access and can add comments.
There's no hard limit on the free plan. Invite as many collaborators as your project needs.

Export & Data

DOCX (Word) export is available now on all plans. PDF export is planned for Pro. The DOCX output uses proper screenplay formatting — Courier 12pt, correct margins, page numbers, and optional scene numbers.
You can paste plain text into a new project and use the element type buttons to format it. Native import from Final Draft (.fdx) and Fountain (.fountain) formats is on the roadmap.
